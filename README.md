# Alertă ANM Județ România

Integrare Home Assistant pentru alertele meteorologice ANM pe județe.

## Instalare

### Prin HACS (Recomandat)
1. Deschide HACS în Home Assistant
2. Adaugă acest repository ca repository custom
3. Caută "Alertă ANM Județ România"
4. Instalează

### Manual
1. Copiază folderul `alerta_anm_judet_romania` în directorul `custom_components` din Home Assistant
2. Restart Home Assistant

## Configurare

1. Mergi la **Settings** → **Devices & Services**
2. Click pe **Add Integration**
3. Caută "Alertă ANM Județ România"
4. Urmează pașii de configurare
